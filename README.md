# Julián Morales — Tattoo Artist Website

Personal website for **Julián Morales**, Black & Gray Realism tattoo artist based at Premium Clan Tattoo Studio, San Jose, CA.

## Stack

- [Next.js 14](https://nextjs.org/) (App Router) + TypeScript
- [Tailwind CSS](https://tailwindcss.com/) — gold & black brand palette
- [Framer Motion](https://www.framer.com/motion/) — section animations
- [React Hook Form](https://react-hook-form.com/) — contact form
- [Lucide](https://lucide.dev/) — icons

## Sections

1. **Hero** — name, specialty, primary CTA
2. **Trust strip** — verified IG, studio, healed pieces, bilingual
3. **About** — bio + specialty
4. **The Process** — 4-step booking journey
5. **Portfolio** — placeholder grid (real images pending)
6. **Guarantee** — 90-day touch-up commitment
7. **FAQ** — pricing, deposit, reschedule, walk-ins, etc.
8. **Contact** — form with A2P-compliant SMS consent checkboxes
9. **Footer** — social, legal links, location

## Pages

- `/` — main landing
- `/privacy` — Privacy Policy (A2P 10DLC compliant)
- `/terms` — Terms of Service (A2P 10DLC compliant)

## Dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

- Auto-deploy via Vercel on push to `main`.
